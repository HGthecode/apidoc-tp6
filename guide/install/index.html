# 安装/更新

在安装本插件时，确保你已成功安装ThinkPHP6的项目并成功运行

安装方法参考：ThinkPHP6文档 (opens new window)

# 安装插件

进入ThinkPHP6 项目根目录，执行如下命令：

composer require hg/apidoc
1

# 添加前端页面

前端版本(点击下载) 与apidoc版本对照
v1.3.0 (opens new window) 适用于v2.5.0版本
v1.2.1 (opens new window) 适用于v2.4.2版本
v1.2.0 (opens new window) 适用于v2.4.0 - v2.4.1版本
v1.1.0 (opens new window) 适用于v2.0.0 - v2.3.0版本
v1.0.0 (opens new window) 适用于v1.x 版本

下载完成后解压，将apidoc文件夹拷贝到你的项目 public 目录下

安装插件完成后 打开浏览器访问 http://你的域名/apidoc/ ，出现接口文档页面，表示安装成功。

接下来，了解一下 基本配置 和 如何使用 吧。

如遇页面报错或无法正常使用，可参考常见问题

# 升级插件

1、如果你之前已经安装过，那么切换到你的应用根目录下面，然后执行下面的命令进行更新。

composer update hg/apidoc
1

2、更新前端页面，可通过安装时的添加前端页面 方式获得最新的前端文件，覆盖/public/apidoc的文件夹即可。
