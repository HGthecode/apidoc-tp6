# 安装

在安装本插件时，确保你已成功安装ThinkPHP的项目并成功运行

安装方法参考：ThinkPHP文档 (opens new window)

# 安装插件

进入ThinkPHP 项目根目录，执行如下命令：

composer require hg/apidoc

# 添加前端页面

# 方式一 (推荐)

点击下载 (opens new window) ，下载完成后解压，将apidoc文件夹拷贝到你的项目 public 目录下

# 方式二 GitHub 下载

1、执行以下命令下载UI

git clone https://github.com/HGthecode/apidoc-ui.git

2、下载完成后，将apidoc目录放到你的项目public目录下

安装插件完成后 打开浏览器访问 http://你的域名/apidoc/ ，出现接口文档页面，表示安装成功。

TP5.x版本用户出现访问页面接口404的情况，请查看 TP5.x版本文档接口404

接下来，了解一下 基本配置 和 如何使用 吧。

# 升级插件

1、如果你之前已经安装过，那么切换到你的应用根目录下面，然后执行下面的命令进行更新。

composer update hg/apidoc

2、更新前端页面，可通过安装时的添加前端页面 方式获得最新的前端文件，覆盖/public/apidoc的文件夹即可。
