# 更新日志

#### 发布周期
- 主版本号：含有破坏性更新和新特性，不在发布周期内。
- 次版本号：带有新特性的向下兼容的版本。
- 修订版本号：日常 bugfix 更新


### v2.1.1
`2020-02-11`

- 新增可视化一键Crud
- 全新的多应用、多版本管理
- 优化支持Responses统一响应体返回配置，[responses配置](/v2/config/#responses)
- 支持配置读取数据表字段时自动转驼峰
- 修正tag显示问题
